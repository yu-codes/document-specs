# 參、管理者操作手冊

## 3.1 管理功能概述

| 管理功能 | 說明 | 適用角色 |
|---------|------|---------|
| 使用者管理 | 帳號建立、停用、權限指派 | 系統管理者 |
| 角色管理 | 角色定義、權限設定 | 系統管理者 |
| 系統設定 | 全域參數設定 | 系統管理者 |
| 日誌查看 | 操作日誌、系統日誌 | 系統管理者 |
| 報表管理 | 報表產出、排程 | 管理者 |
| 通知管理 | 通知範本、發送設定 | 管理者 |

## 3.2 使用者管理

### 新增使用者

| 步驟 | 操作 | 說明 |
|------|------|------|
| 1 | 進入「使用者管理」 | 左側選單 → 系統管理 → 使用者管理 |
| 2 | 點選「新增」 | 右上角新增按鈕 |
| 3 | 填寫使用者資料 | 姓名、Email、部門、角色 |
| 4 | 設定初始密碼 | 或選擇系統自動產生 |
| 5 | 儲存 | 系統自動發送通知信 |

### 使用者資料欄位

| 欄位 | 必填 | 說明 |
|------|------|------|
| 姓名 | ✓ | 使用者姓名 |
| Email | ✓ | 登入帳號 |
| 部門 | ✓ | 所屬部門 |
| 角色 | ✓ | 系統角色（可多選） |
| 電話 | ○ | 聯絡電話 |
| 狀態 | ✓ | 啟用/停用 |

### 停用/啟用帳號

| 步驟 | 操作 |
|------|------|
| 1 | 搜尋目標使用者 |
| 2 | 點選「停用」或「啟用」 |
| 3 | 確認操作 |

### 重設密碼

| 步驟 | 操作 |
|------|------|
| 1 | 搜尋目標使用者 |
| 2 | 點選「重設密碼」 |
| 3 | 選擇重設方式（手動指定/自動產生/發送重設連結） |
| 4 | 確認並通知使用者 |

## 3.3 角色與權限管理

### 預設角色

| 角色 | 權限範圍 | 說明 |
|------|---------|------|
| ADMIN | 所有功能 | 系統最高權限 |
| MANAGER | 部門管理 + 報表 | 部門管理者 |
| USER | 基本操作 | 一般使用者 |
| VIEWER | 唯讀 | 僅能檢視 |

### 自訂角色

| 步驟 | 操作 |
|------|------|
| 1 | 進入「角色管理」 |
| 2 | 點選「新增角色」 |
| 3 | 設定角色名稱與描述 |
| 4 | 勾選權限項目 |
| 5 | 儲存 |
