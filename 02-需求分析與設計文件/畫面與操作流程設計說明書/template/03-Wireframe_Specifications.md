# 柒、畫面元素說明與互動行為

## 7.1 通用元件規範

### 按鈕（Button）

| 類型 | 使用情境 | 樣式 | 狀態 |
|------|---------|------|------|
| Primary | 主要行動 | 填滿主色 | default/hover/active/disabled/loading |
| Secondary | 次要行動 | 邊框主色 | default/hover/active/disabled |
| Danger | 破壞性操作 | 填滿紅色 | default/hover/active/disabled |
| Ghost | 低優先級 | 僅文字 | default/hover/active/disabled |

### 表單元件

| 元件 | 狀態 | 行為 |
|------|------|------|
| Input | default/focus/error/disabled | 即時驗證 |
| Select | default/open/selected/disabled | 下拉選單 |
| Checkbox | unchecked/checked/indeterminate | 多選 |
| Radio | unselected/selected | 單選 |
| Toggle | off/on | 即時生效 |

### 回饋元件

| 元件 | 觸發情境 | 持續時間 | 位置 |
|------|---------|---------|------|
| Toast | 操作成功/失敗 | 3 秒自動消失 | 右上角 |
| Modal | 確認操作/重要訊息 | 使用者關閉 | 畫面中央 |
| Alert | 表單驗證錯誤 | 修正前持續 | 欄位下方 |
| Loading | 資料載入 | 載入完成 | 內容區域 |

## 7.2 互動行為規範

| 互動 | 觸發 | 效果 | 持續時間 |
|------|------|------|---------|
| 頁面切換 | 導覽點擊 | Fade transition | 300ms |
| Modal 開啟 | 按鈕點擊 | Fade + Scale | 200ms |
| Dropdown 展開 | 點擊/Focus | Slide down | 150ms |
| 刪除確認 | 刪除按鈕 | 顯示確認 Modal | — |
| 表單送出 | Submit | Loading → Success/Error | — |

## 7.3 錯誤狀態設計

| 情境 | 顯示方式 | 文案範例 | 回復動作 |
|------|---------|---------|---------|
| 欄位驗證錯誤 | 紅色邊框 + 錯誤訊息 | 「請輸入有效的電子郵件」 | 修正後自動清除 |
| 伺服器錯誤 | Toast / Alert | 「系統暫時無法處理，請稍後再試」 | 提供重試按鈕 |
| 網路中斷 | 全頁提示 | 「網路連線中斷」 | 自動重連 |
| 404 頁面 | 自訂 404 頁面 | 「找不到此頁面」 | 返回首頁按鈕 |
| 權限不足 | 提示訊息 | 「您沒有存取此功能的權限」 | 返回上一頁 |
