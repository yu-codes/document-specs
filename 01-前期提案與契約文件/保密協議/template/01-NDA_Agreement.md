## 貳、保密定義與範圍

### 2.1 機密資訊定義

「機密資訊」係指任一方（揭露方）向另一方（接收方）揭露或提供之所有非公開資訊，包括但不限於：

| 類別 | 說明 | 範例 |
|------|------|------|
| 商業資訊 | 營運策略、客戶名單、商業計畫 | [填寫] |

| 其他 | 雙方書面認定之其他機密資訊 | [填寫] |

### 2.3 機密資訊形式

機密資訊包含以下任何形式之揭露：

- 書面或印刷文件
- 電子郵件或數位檔案
- 口頭或視覺展示
- 軟體、程式碼及相關文件
- 原型、樣品或模型
- 其他任何有形或無形之形式

### 2.4 排除條款

以下資訊不屬於本協議所定義之機密資訊：

| 排除條件 | 說明 |
|---------|------|
| 公開資訊 | 接收方收到時已屬公開或其後非因接收方之過失而成為公開者 |
| 自有資訊 | 接收方能證明在揭露前已合法持有之資訊 |
| 獨立開發 | 接收方能證明係獨立開發，未參考機密資訊者 |
| 第三方合法取得 | 接收方自無保密義務之第三方合法取得者 |
| 書面同意揭露 | 經揭露方書面同意揭露者 |

## 參、保密義務與限制

### 3.1 保密義務

接收方應：

1. 以不低於保護自身機密資訊之同等注意程度保護揭露方之機密資訊
2. 僅在本協議約定之目的範圍內使用機密資訊
3. 限制機密資訊之接觸人員，僅限因業務需要而必須知悉者
4. 確保接觸機密資訊之人員均知悉並遵守本協議之保密義務
5. 未經揭露方書面同意，不得複製、翻印或以其他方式重製機密資訊

### 3.2 使用限制

| 限制項目 | 說明 |
|---------|------|
| 使用目的 | 僅限於 [填寫專案/合作目的] |
| 接觸人員 | 僅限本專案相關人員（名單另附） |
| 儲存方式 | [填寫：加密儲存/指定區域] |
| 傳輸方式 | [填寫：加密傳輸/安全通道] |
| 複製限制 | 未經書面同意不得複製 |
| 外部揭露 | 嚴禁向任何第三方揭露 |

### 3.3 許可揭露

在以下情況下，接收方得揭露機密資訊：

1. 經揭露方事先書面同意
2. 法律、法規或法院命令要求揭露（應事先通知揭露方）
3. 向其專業顧問（律師、會計師）揭露（該等顧問須受同等保密義務約束）

### 3.4 員工與承包商管理

- 接收方應確保其員工及承包商簽署個別保密承諾
- 接收方對其員工及承包商之違約行為負連帶責任
- 接觸機密資訊之人員清單應於簽署後 [填寫] 日內提供

## 伍、違約責任與救濟

### 5.1 違約定義

以下行為構成違約：

- 未經授權揭露機密資訊予第三方
- 超出約定目的使用機密資訊
- 未依約定保管或銷毀機密資訊
- 未確保人員遵守保密義務
- 其他違反本協議條款之行為

### 6.5 可分割性

本協議任一條款如經認定為無效或不可執行，不影響其餘條款之效力。

### 6.6 轉讓

未經他方書面同意，任一方不得將本協議之權利或義務轉讓予第三方。

### 6.7 拋棄
